Get Course Description

This is a POST request that retrieves course details

    Paramaters:
  • action: name of function to be called in the api (for example 'POST' is getDescription)
  • getDescription: the id of the course in the Course table. holds the subject area i.e: CIS and level of the course i.e: 3760.
      • POST https://cis3760f23-08.socs.uoguelph.ca./api.php

      Body urlencoded

      action getDescription
      getDescription CIS*1050

      Example Response:

              Add Course

              This is a POST request that adds a new course

                Paramaters:
            • action: name of function to be called in the api (for example 'POST' is addCourse)
            • courseID: the ID of the course
            • courseName: the name of the course
            • credit: the weight of the course
            • semester: fall, winter, spring or all year
            • description: the description of the course
                • POST https://cis3760f23-08.socs.uoguelph.ca./api.php

                Body urlencoded

                action addCourse
                courseID RTX*3070
                courseName Graphic Card
                credit 0.5
                semester fall
                description test course
